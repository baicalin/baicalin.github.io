Condition Spec

A condition spec defines a simulation request in a stable, repeatable structure.

Required Fields

  • dataset_id: identifier of prepared data.
  • mode: perturbation mode such as knockdown.
  • targets: list of genes or proteins to perturb.
  • condition: contextual label for the scenario.

Example Spec

{
  "dataset_id": "ds_example_001",
  "mode": "knockdown",
  "targets": ["GENE_A", "GENE_B"],
  "condition": "treated"
}
