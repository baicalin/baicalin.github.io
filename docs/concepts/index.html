Concepts

These concepts provide a shared vocabulary for design, execution, and review.

Perturbation Mode

Defines how targets are modified in simulation. Typical modes are knockdown and activation.

Condition

Represents experimental context (for example control vs treated) where target effects are estimated.

Output Interpretation

  • Prioritize candidates by effect magnitude and consistency.
  • Review marker-level behavior aligned to biological rationale.
  • Use exports for downstream ranking and experiment planning.
