API

Use the API for repeatable, scriptable simulation runs.

Authentication

Send an API key in the Authorization header as Bearer <YOUR_API_KEY>.

Endpoint

  • POST /v1/simulations: create and run a simulation request.

Minimal Request

curl -X POST "https://api.knockg.com/v1/simulations" \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "dataset_id": "ds_example_001",
    "mode": "knockdown",
    "targets": ["GENE_A", "GENE_B"],
    "condition": "treated"
  }'

Response (shape)

{
  "simulation_id": "sim_12345",
  "status": "completed",
  "summary": {
    "top_candidates": ["GENE_A", "GENE_B"]
  }
}
