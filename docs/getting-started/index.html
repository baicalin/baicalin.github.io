Getting Started

Use this page to run your first simulation and validate output quality quickly.

Checklist

  • Dataset is prepared with baseline and treated conditions.
  • Target genes or proteins are selected for perturbation.
  • Success criteria for the first run are defined.

First Run Flow

  1. Create a new simulation and choose perturbation mode.
  2. Configure condition parameters and selected targets.
  3. Run simulation, inspect marker-level changes, and export summary.
