Regular license can be used for end products that do not charge users for access or service. Single regular license can be used for a single end product.

An extended license allows for an end product that charges users for access. It can be used for multiple domains with the same end product.

The extended license is applicable for SaaS applications as it allows you to charge users for access to the application.
